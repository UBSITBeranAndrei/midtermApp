Data Binding

String Interpolation

A one-way data binding technique that displays dynamic data from your TypeScript component using double curly braces {{ curly }}.

Basic Characteristics
  • ✅ Component Property: {{ common.username }}
  • ✅ Arithmetic: {{ 1 + 1 }}
  • ✅ Methods: {{ getWelcomeMessage() }}
  • ✅ String Methods: {{ title.toUpperCase() }}
Rules & Constraints
  • ✅ Must return a value
  • ✅ Can call pure methods and use pipes
  • ❌ Cannot use assignments (=) or (++)
  • ❌ Cannot access window or document
